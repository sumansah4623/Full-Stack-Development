What is Express.js?

Express.js is a framework for Node.js that helps us build backend servers and APIs easily.

In simple words:
Express = easy way to create a server in Node.js



Basic Express App
const express = require("express");
const app = express();

app.get("/", (req, res) => {
    res.send("Hello World");
});

app.listen(7000, () => {
    console.log("Server running");
});

npm init
npm i express
npm start
